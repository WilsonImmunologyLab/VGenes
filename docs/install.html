Installation Instructions

  • Install VGenes on MAC OS (Click to show/hide)

    Install VGenes

    • Download DMG file from the link on the top
    • Double click the DMG file and then drag and drop "VGenes" to "Applications" folder
    • Then please go to your application and click VGenes to open
    • You may see this when you try to open VGenes:
      • or
      Don't worry, please just click "Cancel" and then open your terminal (click here for how to open terminal), and then type

      xattr -cr /Applications/VGenes.app

      Then you're good to go!
    • Users can make a alias on desktop for better(faster) user experience by the following steps:

    Install Dependencies

    We already included Muscle, Clustal Omega and RAxML in VGenes. If those pre-complied softwares don't work on your computer, you can download or complie from offical website and determine the path for those tools in Setting

    Install Dependencies - optional

    GhostScript and Pdf2svg (Recommended, for a faster sequence logo)

    Install Home brew. It's a Missing Package Manager for macOS (or Linux). You may need the help form homebrew to install some missing packages. Open a terminal and paste the follow commands, then enter

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

    After you installed home brew, you can install the following packages for better user experience.

    brew install ghostscript

    brew install pdf2svg



  • Install VGenes on Windows (Click to show/hide)

    Install VGenes

    • Download zipped executable file from the link on the top
    • Unzip the zipped file and put VGenes folder to a local folder you prefer
    • Save User Guide to your computer for reference
    • Users can make a shortcut on desktop for better (faster) user experience.

    Install Dependencies

    We already included Muscle, Clustal Omega and RAxML in VGenes. If those pre-complied softwares don't work on your computer, you can download or complie from offical website and determine the path for those tools in Setting

    Install Dependencies - optional

    GhostScript (Recommended, for a faster sequence logo)

    Please download and install GhostScript from this link, and set up system PATH according to the guidence in our user guide.
